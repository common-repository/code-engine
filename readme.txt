== Description ==

Code Engine helps you manage code snippets from Code Snippets and WP Code. It checks errors, gathers snippets, and creates JSON for AI models.

It's experimental and not ready for production. It's a tool for developers who want to play with functions that can be used in AI models. 

== Installation ==

1. Upload `code-engine` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
